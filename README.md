# JNCT_Mock_Test_Website
🎓 JNCT Mock Test Website (Ultimate v12)

A fully functional Mock Test Web Application designed for students and teachers.
This project allows teachers to create, manage, and organize questions, while students can attempt tests in a real exam-like environment.

---

🚀 Features

👨‍🏫 Teacher Panel

- Secure password-protected access
- Add, edit, and delete questions
- Multiple-choice question support (4 options)
- Mark correct answers easily
- Save questions into folders (Question Bank)
- Create, rename, and delete folders
- Import questions from saved bank
- Set test duration (timer control)
- Customize subject name and logo
- Generate printable PDF question paper
- Save all data in browser (Local Storage)

---

👨‍🎓 Student Panel

- Start test with confirmation
- Timer-based exam system ⏱️
- Navigate between questions
- Question navigation panel (attempt tracking)
- Auto-submit when time ends
- Instant result calculation
- Answer review (with teacher password)

---

💾 Data Storage

- Uses LocalStorage (no backend required)
- Stores:
  - Questions
  - Settings (subject, logo, password, timer)
  - Question bank folders

---

🖥️ Technologies Used

- HTML5
- CSS3 (Modern UI Design)
- JavaScript (Vanilla JS)
- Browser LocalStorage API

📄 How to Use

For Teachers:

1. Click on Teacher Panel
2. Enter password
3. Add questions and options
4. Set correct answers
5. Save or organize questions in folders
6. Set timer and subject
7. Save all data

For Students:

1. Click on Students
2. Start test
3. Attempt all questions
4. Submit test
5. View score instantly
